# CustomerCRUDApp
HTTP API server with basic CRUD operations for the Customer entity
## Tech stack
+ **Java 17**
+ **Spring Boot, Data JPA, Web**
+ **Hibernate**
+ **PostgreSQL**
+ **Swagger**
+ **Maven**
+ **Postman**
